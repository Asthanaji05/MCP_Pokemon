
# Characteristic

Characteristics indicate which stat contains a Pokémon's highest IV. A Pokémon's Characteristic is determined by the remainder of its highest IV divided by 5.
Endpoint: [https://pokeapi.co/api/v2/characteristic/](https://pokeapi.co/api/v2/characteristic/)
